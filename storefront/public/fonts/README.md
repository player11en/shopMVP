# Fonts Directory

## Tonnelier Font

To use the custom Tonnelier font, place the font files here:

- `Tonnelier-Regular.woff2`
- `Tonnelier-Regular.woff`

### Where to get Tonnelier font:

1. If you have the font files, place them in this directory
2. Or use a web font service like Google Fonts
3. Or update `globals.css` to use a different font

### Fallback:

If the font files are not found, the site will use `sans-serif` as a fallback.
